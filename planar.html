Planar Mobile Robot Modeling

Modeled the kinematics and developed autonomous navigation controls for a physical differential-drive robot to accurately navigate to user-selected points.

Kinematic Modeling Autonomous Navigation Control Systems Matlab to Hardware Integration

1. Overview

Modeled the kinematics of a differential-drive planar mobile robot and developed a control system tailored for autonomous navigation. The primary focus was enabling the physical robot system to accurately and smoothly navigate to any user-selected target point in its environment.

The project successfully bridged the gap between theoretical mathematical modeling and real-world physical hardware execution.

2. Problem Statement

Autonomous navigation of a planar mobile robot requires accurate modeling of its kinematics and robust control algorithms to handle real-world physical constraints. The core challenge was translating a simulated model into reliable physical movement.

The objective was to:

  • Formulate the kinematic model of a differential-drive robot
  • Develop closed-loop control algorithms for point-to-point navigation
  • Translate simulated math models to control a real physical robot

3. Kinematic Modeling

I developed a comprehensive mathematical model representing the non-holonomic constraints of the differential-drive system. This involved deriving the forward and inverse kinematics to map individual wheel velocities to the robot's global position and orientation.

This accurate kinematic model formed the rigid foundation required for the subsequent autonomous control design.

4. Control Strategy for Autonomous Navigation

To achieve reliable autonomous navigation to user-selected points, I implemented a robust closed-loop control strategy. The custom controller continuously calculates the required linear and angular velocities based on the error between the robot's real-time pose and the target destination.

6. Experimental Validation

The navigation system was extensively tested on the physical robot. Experiments involved providing various user-selected target points across different distances, angles, and real-world floor surfaces.

The results successfully validated the accuracy of the kinematic model and the reliability of the control algorithms, with the robot consistently reaching its designated targets with high precision.

7. My Contributions

  • Modeled the forward and inverse kinematics for the differential-drive robot
  • Developed and tuned closed-loop control algorithms for point-to-point autonomous navigation
  • Bridged theoretical mathematical models to control real-world physical robot hardware
  • Validated autonomous navigation accuracy through extensive physical testing
